# 🌍 全球 AI 日报 | 2026-02-18

## 今日核心判断

OpenAI 和 Google 在推理模型上的军备竞赛进入白热化——GPT-5.2 主打长任务可靠性，Gemini 3 Flash 则用价格屠刀抢占 API 市场。对开发者来说，推理成本在三个月内又降了 40%，Agent 应用的经济可行性正在快速改善。与此同时，Anthropic 悄悄押注企业级 workflow 自动化（Cowork + Knowledge Bases），走的是和 OpenAI 完全不同的商业化路径。

## 🚀 重大发布与产品

**OpenAI 发布 GPT-5.2，主打 Agent 场景长时运行稳定性**
GPT-5.2 重点优化了多步骤任务的上下文保持能力，官方称在 30 分钟以上的 coding agent session 中完成率提升 35%。定价与 GPT-5 持平。这直接回应了 Gemini 3 发布后开发者社区对 OpenAI 竞争力的质疑，信号很明确：OpenAI 认为 Agent 可靠性是当前主战场。

**Anthropic 为 Cowork 引入 Knowledge Bases 功能**
用户可创建持久化知识库供 Claude 在任务中引用和增量更新。这意味着 Anthropic 的企业产品从"对话助手"转向"带记忆的工作流引擎"，直接对标 Microsoft Copilot 的企业知识管理场景。

**Google Stitch 新增 API Key 管理和 PRD 自动生成**
Google 的设计→开发工具 Stitch 开始覆盖产品经理工作流。结合 Gemini 模型生成高分辨率设计稿和项目文档，Google 在"AI 原生开发工具链"上的布局越来越完整。

## 🔬 技术与研究

**NVIDIA 展示 CLI Agent 训练方法：合成数据 + RL，零先验知识学会运维**
用合成数据和可验证奖励的 RL 训练推理模型操作命令行，能完成 Docker 部署、服务器启动等复杂任务。关键启示：Agent 不一定需要从互联网数据中学——合成数据 + RL 可能是垂直领域 Agent 的最优路径。

**FlashAttention 团队发布 v3.1，A100 吞吐量再提 22%**
Tri Dao 团队持续压榨 GPU 性能，新版本在 A100 上实现 22% 吞吐提升，H100 上提升 15%。这类底层优化直接降低所有模型的推理成本，是整个行业的"水电煤"。

## 💰 融资与市场

**EntireHQ 完成 $60M 种子轮，做 Git 兼容的 AI 代码知识库**
由 Sequoia 领投。产品定位是在代码仓库之上构建一层"意图和上下文"数据库，供 AI Agent 理解代码背景。赛道逻辑：当 coding agent 成为标配，理解代码库上下文的基础设施就是刚需。

**CoreWeave 据报正筹备 IPO，估值或超 $35B**
GPU 云服务商 CoreWeave 的 IPO 进程加速，反映市场对 AI 基础设施需求的持续看好。作为 NVIDIA 最大的云客户之一，其估值也是 AI 算力需求的风向标。

## 💎 芯片与算力

**SK Hynix 宣布 HBM4 量产提前至 Q3 2026**
原计划 Q4 量产，提前一个季度。HBM4 带宽较 HBM3e 翻倍，直接影响下一代 GPU（B300/MI400）的上市节奏。对 AI 训练和推理的成本曲线是实质利好。

**Broadcom 发布新一代 AI ASIC 设计平台**
面向 Google TPU、Meta MTIA 等定制芯片客户，提供 3nm 工艺支持。ASIC 定制化趋势进一步确认：头部公司在特定工作负载上不再满足于通用 GPU。
